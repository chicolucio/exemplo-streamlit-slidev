# O que é Streamlit?

Pacote Python que facilita a criação de páginas web interativas. Muito usado por
cientistas de dados.

- 📝 **Markdown**
- 🎨 **Temas**
- 🧑‍💻 **Python**
- 🤹 **Interativo**
- 🎥 **Bibliotecas gráficas** - Matplotlib, Plotly, Bokeh
- 📤 **Bibliotecas científicas** - NumPy, Pandas, Scikit-Learn
- 🛠 **Tipografia** - LaTeX
